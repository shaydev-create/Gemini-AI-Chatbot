=== CÓMO TOMAR LAS CAPTURAS ===

🔧 **Preparación:**
1. Instalar la extensión en modo desarrollador
2. Configurar con una API Key válida
3. Tener algunas conversaciones de ejemplo
4. Usar un navegador limpio (sin otras extensiones visibles)

📱 **Herramientas Recomendadas:**
- Chrome DevTools (F12) para ajustar tamaño de ventana
- Snipping Tool de Windows
- OBS Studio para capturas de alta calidad
- Canva o Figma para capturas promocionales

🎨 **Consejos de Diseño:**
- Usar fondos limpios y profesionales
- Asegurar que el texto sea legible
- Mostrar la funcionalidad real de la extensión
- Evitar información personal o API Keys reales
- Usar datos de ejemplo atractivos
