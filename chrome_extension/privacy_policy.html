🔒 Política de Privacidad

🚀 Gemini AI Futuristic Chatbot

Última actualización: 17 de enero de 2025

🛡️ Compromiso de Privacidad: Esta extensión de Chrome respeta completamente tu privacidad. No recopilamos, almacenamos ni compartimos ningún dato personal. La extensión solo facilita el acceso a tu servidor local de Gemini AI.

1. 📋 Información que NO Recopilamos

Esta extensión está diseñada con la privacidad como prioridad. NO recopilamos, almacenamos ni transmitimos ningún tipo de información personal, incluyendo:

  • ❌ Datos personales (nombre, email, teléfono, dirección)
  • ❌ Conversaciones o mensajes del chat (solo se procesan localmente)
  • ❌ Archivos o documentos subidos
  • ❌ Historial de navegación
  • ❌ Ubicación geográfica
  • ❌ Información de identificación del dispositivo
  • ❌ Cookies o datos de seguimiento

2. ⚡ Cómo Funciona la Extensión

Esta extensión funciona como un cliente independiente para la API de Google Gemini:

  • 🔗 Se conecta directamente a la API de Google Generative Language
  • 🔑 Utiliza tu API Key personal de Google (almacenada localmente)
  • 💻 Las conversaciones se procesan en tiempo real sin almacenamiento
  • 🔐 Comunicación directa con Google sin intermediarios
  • 🗂️ Solo almacena tu API Key de forma segura en Chrome Storage

2.1. 🔑 Uso de API Key de Google

Para funcionar, la extensión requiere que proporciones tu propia API Key de Google Gemini:

  • 🔒 Tu API Key se almacena únicamente en tu navegador (Chrome Storage Sync)
  • 🚫 Nunca compartimos, vendemos o transmitimos tu API Key a terceros
  • 🗑️ Puedes eliminar tu API Key en cualquier momento desde la configuración
  • 📋 Solo se usa para autenticar tus solicitudes con Google Gemini
  • 🔐 Google maneja todas las solicitudes según su propia política de privacidad

3. 🔐 Permisos Requeridos

La extensión solicita únicamente los permisos esenciales para su funcionamiento:

  • 🔹 activeTab: Para interactuar con la pestaña activa cuando sea necesario
  • 🔹 storage: Para almacenar de forma segura tu API Key en Chrome Storage
  • 🔹 host_permissions (generativelanguage.googleapis.com): Para conectar con la API de Google Gemini

Estos permisos son los mínimos necesarios y no permiten acceso a datos sensibles de navegación.

4. 🛡️ Seguridad y Privacidad

Implementamos las mejores prácticas de seguridad:

  • 🔒 Comunicación segura: Todas las conexiones usan HTTPS
  • 🚫 Sin tracking: No utilizamos cookies ni sistemas de seguimiento
  • 💾 Almacenamiento local: Solo tu API Key se guarda localmente en tu navegador
  • 🔐 Código transparente: Funcionalidad clara y sin código malicioso
  • 🗑️ Control total: Puedes eliminar todos los datos almacenados en cualquier momento

5. 🤖 Procesamiento de IA

Cuando usas el chatbot:

  • 🏠 Servidor local: Tu aplicación Flask procesa las consultas localmente
  • 🔗 API de Google: Tu servidor se conecta directamente a Google Gemini API
  • 🚫 Sin intermediarios: La extensión no intercepta ni modifica datos
  • 🔒 Tu API Key: Solo tú tienes acceso a tu clave de API

5. 🍪 Cookies y Almacenamiento

  • ❌ Sin cookies: La extensión no utiliza cookies
  • ❌ Sin almacenamiento local: No guardamos datos en tu navegador
  • ❌ Sin tracking: No rastreamos tu comportamiento

6. 🌐 Datos de Terceros

La extensión no comparte datos con terceros porque:

  • 🚫 No recopila datos para compartir
  • 🏠 Funciona localmente en tu computadora
  • 🔒 Conexión directa entre tu navegador y tu servidor

7. 👤 Tus Derechos

Como no recopilamos datos personales:

  • ✅ Control total: Tienes control completo sobre tus datos
  • ✅ Sin datos que eliminar: No hay información personal que borrar
  • ✅ Desinstalación fácil: Puedes eliminar la extensión en cualquier momento

8. 🔄 Actualizaciones de la Política

Si realizamos cambios a esta política:

  • 📢 Notificación: Se actualizará la fecha de modificación
  • 🔍 Transparencia: Los cambios serán claramente indicados
  • 📝 Historial: Mantendremos un registro de cambios
